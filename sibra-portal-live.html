⚙️ 資料來源設定

📊 Google Sheets — 每日 KPI 主數據
試算表發布 CSV 網址
Sheets → 檔案 → 共用 → 發布到網路 → 選分頁 → CSV → 複製連結

欄位格式（A～G，第一列為標題）：
A 日期（2024-03-01）　B 營收（元）　C 訂單量
D 新增用戶　E 訪客數　F 加入購物車　G 進入結帳
🛍️ 團購表單（GroupBuyForms）— 業績數據
Google Apps Script 中轉網址
⚠️ GroupBuyForms 無公開 API，採用「匯出貼入 Sheets」方式：

步驟：
1. GroupBuyForms 後台 → 訂單管理 → 匯出 Excel/CSV
2. 將資料貼入一份 Google Sheets（欄位對齊一次就好）
3. 那份 Sheets → 檔案 → 共用 → 發布到網路 → CSV → 複製連結貼上

建議欄位格式（自行整理時對應）：
A 訂單時間　B 買家姓名　C 商品名稱　D 數量　E 單價（元）　F 付款狀態
（付款狀態建議統一用：已付款 / 未付款 / 退款）

💬 LINE 官方帳號 — 好友 / 訊息成效
Google Apps Script 中轉網址
因 LINE API 有跨網域限制，需用 Google Apps Script 橋接。
在 script.google.com 新增專案，貼上以下程式後「部署為網頁應用程式」→ 複製網址： const LINE_TOKEN = 'YOUR_CHANNEL_ACCESS_TOKEN'; function doGet() { const date = Utilities.formatDate( new Date(), 'Asia/Taipei', 'yyyyMMdd' ); // 好友數 const fRes = UrlFetchApp.fetch( 'https://api.line.me/v2/bot/followers/ids?limit=1', {headers: {Authorization: 'Bearer ' + LINE_TOKEN}} ); // 訊息成效（需 LINE Official Account Manager API） const iRes = UrlFetchApp.fetch( 'https://api.line.me/v2/bot/insight/message/delivery?date=' + date, {headers: {Authorization: 'Bearer ' + LINE_TOKEN}} ); const followers = JSON.parse(fRes.getContentText()); const insight = JSON.parse(iRes.getContentText()); return ContentService .createTextOutput(JSON.stringify({ followerCount: followers.userIds?.length || 0, delivered: insight.broadcast?.delivered || 0, uniqueImpression: insight.broadcast?.uniqueImpression || 0, openRate: insight.broadcast?.uniqueImpression && insight.broadcast?.delivered ? +(insight.broadcast.uniqueImpression / insight.broadcast.delivered * 100).toFixed(1) : 0 })) .setMimeType(ContentService.MimeType.JSON); }
📈 Google Analytics 4 — 網站流量
Google Apps Script 中轉網址
同樣需要 GAS 橋接（使用你的 Google 帳號授權，不需另外申請 Key）： const GA4_PROPERTY = 'YOUR_PROPERTY_ID'; // 純數字 function doGet() { const body = { dateRanges: [{startDate:'30daysAgo', endDate:'today'}], metrics: [ {name:'sessions'}, {name:'newUsers'}, {name:'screenPageViews'}, {name:'bounceRate'} ] }; const res = UrlFetchApp.fetch( `https://analyticsdata.googleapis.com/v1beta/properties/${GA4_PROPERTY}:runReport`, { method: 'post', contentType: 'application/json', payload: JSON.stringify(body), headers: {Authorization: 'Bearer ' + ScriptApp.getOAuthToken()} } ); return ContentService .createTextOutput(res.getContentText()) .setMimeType(ContentService.MimeType.JSON); }
